# Attributes

| Name |
|-|
|DashStyle|
|FontColor|
|OutlineColor|
|ShowNonOutlineText|
|StrokeWidth|
|Text|

# Example

```xml
<wuc:OutlineTextControl Height="200"
                        DashStyle="Solid"
                        FontSize="120"
                        OutlineColor="Purple"
                        StrokeWidth="4"
                        Text="WinUICommunity" />
```
